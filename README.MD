# **TXP_PLAYERS**

## **DESCRIPTION :**
Basic commands to manage players.

## **HOW TO CONFIGURE :**
- *Anythings able to be config are in `shared/config.lua`*

## **HOW TO USE :**
- *Command `/tpto <player_id>` to teleport to target player.*
- *Command `/tp <player_id>` to teleport target player to you.*
